== Frequently asked questions ==

= Order Button is not Working =

 Make sure Checkout page template is set to "Elementor checkout".

= Plugin is unable to deactivate =

 Check if both woocommerce and elementor are active.

= Why i cannot move coupon section ? =

Coupon section is a different form than checkout. Hence ,it is kept as default so not disturb the order processing . That's why there is no widget for coupon section which you can move .


== Installation ==

1)Make sure woocommerce and elementor plugin are active before installing or activating the plugin.
2)After activating the plugin go to your checkout page and click edit .
3)Make sure the checkout page has selected a template "Elementor Checkout".
4) Remove any kind of shortcode or data from checkout page and click "Edit with elementor" button.
5) In the left widget panel . you will notice 4 new widgets under "woocommerce_checkout" category.
6)Drag your widget from left to right to create your own design .Save all changes.
7)All done.View your checkout page.

All done ! you can view your checkout page or start editing with elementer(Add some products to cart before starting editing).
